# 监控指标

指标名 描述
TTFB 浏览器发起第一个请求到服务端返回第一个字节所消耗的时间
FP （fist paint）首次绘制时间 把一个像素点绘制在屏幕上
FID 用户首次和页面交互和页面响应的时间
卡顿 超过50ms的长任务
PV 页面浏览量点击量
UV 不同Ip的访问人数
停留时间 同一个ip页面停留时间
